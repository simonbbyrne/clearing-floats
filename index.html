Clearing Floats

Tony Aslett & Nicholas Gallagher recommended way of clearing floats

When we float an element, the browser takes that element from the document flow and pushes it to the edge of its parent container.

HTML

            
                <article class="group">
                    <img src="images/gnasher.jpg" />
                    <p>When we float an element, the browser takes .</p>
                </article>

CSS

            
                .group:before,
                .group:after {
                    content: "";
                    display: table;
                }

                .group:after {
                    clear:both;
                }

                .group {
                    zoom: 1; /* IE6/IE7 */
                }
